# 術語統一表

| 術語 | 統一用法 | 備註 |
|------|----------|------|
| Level 1 | Level 1：模擬入門 | 不寫「初級」或「基礎班」 |
| Level 2 | Level 2：AP/IB/A-Level 專題 | 不單獨寫 AP 或 IB，三個課綱並列 |
| Level 3 | Level 3：數值方法與優化 | 不寫「進階 Python」 |
| Level 4 | Level 4：Research / Capstone | 斜線保留，不寫「研究班」 |
| 1-on-1 | 1-on-1（個別指導） | 不寫「個別課」或「家教時間」 |
| Research Question | Research Question（RQ） | 中文版可寫「研究問題（RQ）」 |
| Proposal | 專題計畫書（Proposal） | 括號保留英文 |
| Rubric | 評分規準（rubric） | 小寫，不寫「評分表」 |
| Portfolio | 作品集（portfolio） | 括號保留英文 |
| Capstone | Capstone 專題 | 不翻譯 |
| Literature Review | 文獻回顧（literature review） | 括號保留英文 |
| Gap | 研究缺口（gap） | 括號保留英文 |
| simulate_once | `simulate_once` | 程式函數名，固定以 code 格式呈現 |
| SIR 模型 | SIR 模型 | 不展開為「易感–感染–康復模型」 |
| ODE | ODE（常微分方程） | 括號加中文說明 |
| NumPy | NumPy | 大小寫固定 |
| SciPy | SciPy | 大小寫固定 |
| AP Research | AP Research | 不縮寫為 APR |
| IB IA | IB IA | Internal Assessment 縮寫 |
| A Level | A Level | 不寫 A-Level（除非在表格中需對齊） |
