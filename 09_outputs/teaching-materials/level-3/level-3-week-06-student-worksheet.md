# Level 3 週 06：高級專題選題與方法設計

## 本週是什麼？

這週是 Level 3 的**中期展示週**。你需要在課堂上向全班介紹你的高級專題方向，並在課後繳交一份完整的「高級專題計畫書」。

這不是期末報告，而是**方向確認**。目的是讓老師和同學幫你檢查：
- 問題有沒有夠具體？
- 你選的工具和數值方法是否合理？
- 你的實驗設計能回答你的問題嗎？

---

## 核心概念：什麼是「高級專題」？

### 與 Level 2 的差別

Level 2 的專題重點是：「設計一個模擬，觀察結果。」

Level 3 的高級專題要多做到：

| 面向 | Level 2 | Level 3 |
|------|---------|---------|
| 模型 | 差分方程、簡單迴圈 | ODE 系統、向量化模型 |
| 工具 | 基礎 Python | NumPy + SciPy |
| 分析 | 跑一次、畫圖 | 收斂測試、敏感度分析 |
| 最佳化 | 沒有或手動調參 | 用 `scipy.optimize` 找最佳解 |
| 報告 | 結果描述 | 方法說明 + 技術限制討論 |

---

## 核心概念一：選題的三個標準

好的 Level 3 高級專題需要同時滿足：

**1. 有清楚的數學/計算問題**
- 不只是「我想模擬 X」
- 而是「我想知道在給定條件下，X 的最佳值是多少」或「X 會如何隨時間演化」

**2. 需要用到 SciPy 的工具**
- 你的問題必須需要 ODE solver、數值積分，或最佳化函數
- 如果只用 for 迴圈就能做，代表還不夠「高級」

**3. 有可以做實驗的維度**
- 你要能改變某個參數，觀察結果如何變化
- 例如：改變初始感染率，看疫情高峰如何變化

---

## 核心概念二：設計方法計畫

一個好的方法計畫包含以下五個部分：

```
問題陳述：我要研究 [什麼問題]，在 [什麼條件/假設] 下。
模型說明：我用 [什麼模型] 來描述這個系統（方程式或文字說明）。
擬用工具：我計畫使用 [具體函數名] 來做 [什麼計算]。
實驗設計：我會固定 [A 參數]，改變 [B 參數]，觀察 [C 指標] 的變化。
預計困難：我預計最難的地方是 [具體技術問題]。
```

## 方法計畫範例

### 範例 A：傳染病模型升級版

```
問題陳述：
我要研究在不同接觸率（β）下，SIR 傳染病模型的感染高峰時間點和規模，
假設總人口固定為 10,000 人，初始感染者 10 人。

模型說明：
SIR 模型：
  dS/dt = -β * S * I / N
  dI/dt =  β * S * I / N - γ * I
  dR/dt =  γ * I
其中 β 為接觸率，γ 為康復率。

擬用工具：
scipy.integrate.solve_ivp（RK45 方法），時間範圍 0–160 天，步長自動選擇。

實驗設計：
固定 γ = 0.1，改變 β 從 0.1 到 0.5（每次增加 0.05），
觀察感染高峰（max(I)）和達到高峰的時間點。

預計困難：
當 β 很大時，感染者增長極快，solve_ivp 可能需要更細的步長；
需要測試 dense_output 參數或調整 max_step。
```

### 範例 B：結構最佳化問題

```
問題陳述：
我要設計一個矩形截面懸臂樑，在承受固定荷重的情況下，
找出使撓度最小且材料用量（體積）不超過限制的最佳截面尺寸。

模型說明：
懸臂樑最大撓度公式：δ = F * L³ / (3 * E * I)
其中 I = b * h³ / 12（矩形截面慣性矩），
決策變數：寬度 b 和高度 h。

擬用工具：
scipy.optimize.minimize（SLSQP 方法），帶入體積限制條件 b * h * L ≤ V_max。

實驗設計：
固定 L = 1m、F = 1000N、E = 200GPa，
改變體積上限 V_max（0.001 到 0.01 m³），
觀察最佳 b/h 比例如何隨 V_max 改變。

預計困難：
目標函數和限制條件的量綱差異大（撓度單位是 mm，體積是 m³），
需要做無因次化或調整縮放。
```

## 今日課堂展示準備

你需要在 5 分鐘內說明以下四點：

1. **我的問題是什麼？**（一句話）
2. **我要用什麼工具？**（具體到函數名）
3. **我打算做什麼實驗？**（改變什麼、觀察什麼）
4. **我預計最大的困難是什麼？**（技術面）

> 提示：說不出「我要用哪個 SciPy 函數」，通常代表方法計畫還不夠具體。
