# 開課前需準備的資料清單

## 一、教學面

### 課程結構與講義
- [ ] 四個 level 的正式課綱（週次表、主題、學習目標）
- [ ] 每週講義簡報（slides）：概念說明 + 範例程式碼 + 練習題
- [ ] 各 level 的「期末產出說明文件」：清楚寫明學生最後要交什麼與評分標準

### 範例與模板
- [ ] 每個 level 至少 2–3 個完整示範專題（程式 + 報告大綱）
  - Level 1：小型模擬玩具（骰子、排隊、簡單 growth）
  - Level 2：可對標 AP/IB/A-level 的完整專題範例
  - Level 3：含 NumPy/SciPy/最佳化的進階專題
  - Level 4：有文獻回顧的研究報告範例
- [ ] 報告模板（Word/Google Docs）：已排好段落標題與格式
- [ ] 簡報模板（Google Slides/PowerPoint）：封面、RQ、方法、結果、討論、結論版型

### 程式與技術備料
- [ ] 共用程式碼範本：
  - `simulate_once` 範例
  - 參數掃描 loop 範例
  - 畫圖模板（折線圖、直方圖、散點圖、heatmap）
- [ ] Level 3 所需套件安裝說明（NumPy、SciPy）與環境設定指南
- [ ] Level 4 文獻搜尋示範（Google Scholar 截圖、範例關鍵字）

### 評量與回饋工具
- [ ] 每個 level 的評分 rubric（簡單表格）：程式正確性、模型設計、分析深度、報告寫作、展示表現
- [ ] 每週小作業／進度 check 表
- [ ] 同儕回饋表（中期與期末展示用）
