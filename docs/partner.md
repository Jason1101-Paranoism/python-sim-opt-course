# 補習班合作方案

## 合作模式

我們提供兩種合作模式，補習班可依空間與招生能力選擇。

---

### 方案 A：場地合作（60/40 分潤）

補習班提供教室與招生管道，教師負責全部教學內容。

| 項目 | 說明 |
|------|------|
| 補習班收益 | 學費 40% |
| 教師收益 | 學費 60% |
| 補習班負責 | 場地、行政、招生宣傳 |
| 教師負責 | 課程內容、教材、批改、個別指導 |
| 最小班級人數 | 4 人開班 |

**試算（以 Level 1，4 人班為例）**：
- 學費收入：4 × NT$30,000 = NT$120,000
- 補習班收益：NT$48,000
- 教師收益：NT$72,000

---

### 方案 B：課程授權（70/30 分潤）

補習班使用我們的教材自行開課，我們提供教材更新與顧問支援。

| 項目 | 說明 |
|------|------|
| 補習班收益 | 學費 70% |
| 授權費 | 學費 30% |
| 補習班負責 | 全部教學、場地、招生 |
| 我們提供 | 完整教材包、評量標準、每季更新、顧問 email 支援 |

---

## 為什麼這套課程適合補習班？

- **差異化競爭**：市面上很少有完整的「Python 建模 + 研究報告」課程
- **高單價客群**：目標學生有 AP/IB 需求或大學申請需求，家長付費意願高
- **課程成果清楚**：每位學生都有報告 + 簡報 + 程式碼，家長看得到進度
- **完整教材包**：不需要從頭開發課程，即插即用

---

## 適合開設的補習班類型

- 有國際課程（AP/IB/A-Level）班的升學補習班
- 有 STEM 或數理班底的補習班
- 已有程式課（例如 APCS）的補習班，想加入進階應用課程
