# 品牌定位

## 一句話定位

**這不是 APCS 語法班，而是 Simulation / Optimization / Research Pathway。**

## 差異化

| 一般 Python / APCS 班 | 本課程 |
|----------------------|--------|
| 教語法、刷題、考試 | 教建模、模擬、研究流程 |
| 產出：程式碼與成績 | 產出：專題報告、研究稿、作品集 |
| 對標競賽（APCS、資奧） | 對標申請（AP Research、IB IA、大學 portfolio） |
| 班級統一進度 | 每週 1-on-1，個人題目量身設計 |

## 價值主張

### 對學生
> 用模擬與優化的方式探索你有興趣的問題，做出一個屬於你自己、說得出來、寫得進申請的研究專題。

### 對家長
> 不只是學 Python，而是把 Python 變成做研究的工具，最終有一份報告和作品集可以用在 AP/IB 評分、大學申請、STEM 競賽。

### 對補習班合作方
> 接在你們現有 Python 班後面的「進階路徑」，幫你留住原本要流出去做專題營隊的學生，同時開出你們產品線裡沒有的高單價課程。

## 不是什麼（Anti-positioning）

- 不是以考試為目的的語法課
- 不是直接幫學生寫專題
- 不是通用 AI / 資料科學課
- 不是快速出證書的線上課程
