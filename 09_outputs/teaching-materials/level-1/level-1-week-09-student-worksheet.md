# Level 1 週 9：小報告寫作練習
## 學生講義（Student Worksheet）

---

## Mini 報告結構

### 問題（Introduction）—— 2–4 句
> 說明：我在模擬什麼？為什麼這個問題值得探究？

```
[題目名稱]

本研究模擬了 [系統名稱]，探究 [某個因素] 對 [某個結果] 的影響。
[一句背景說明，例如：排隊系統廣泛應用於診所、銀行和超市。]
本模擬的研究問題是：[你的研究問題]。
```

---

### 方法（Methods）—— 3–6 句
> 說明：我的模型有什麼假設？用了什麼工具？

```
本模擬使用 Python 的 [用到的工具] 實作 [模型名稱]。
模型假設：[假設 1]；[假設 2]。
實驗設計：固定 [固定的參數]，改變 [變動的參數]，範圍從 [最小值] 到 [最大值]，
共 [N] 組實驗，每組重複 [次數] 次取平均。
```

---

### 結果（Results）—— 每張圖 1–2 句說明
> 貼圖，每張圖下面寫圖說。

```
[貼圖 1]
圖 1：[圖說——這張圖發現了什麼，例如：當服務率提升時，等待時間明顯下降]

[貼圖 2（如有）]
圖 2：[圖說]
```

---

### 結論（Conclusion）—— 2–4 句
> 說明：模擬回答了研究問題嗎？答案是什麼？

```
本模擬發現，[主要結論，例如：服務率是影響等待時間最敏感的參數]。
當 [某條件] 時，[結果]。
這對 [現實情境] 有以下意涵：[1–2 句]。
本模型的主要限制是 [限制]，未來可以改進的方向是 [改進]。
```

---

## 圖說寫法提醒

| 差的圖說 | 好的圖說 |
|---------|---------|
| 「圖 1：折線圖」 | 「圖 1：服務率從 0.1 增至 0.5 時，平均等待時間從 30 分鐘降至 3 分鐘，顯示服務速率的改善效果非常顯著。」 |
| 「圖 2：直方圖」 | 「圖 2：模擬 1000 次後，等待時間大多集中在 0–5 分鐘，但有少數極端值超過 20 分鐘（尾巴較長）。」 |

## 自我檢查清單

- [ ] 問題段說清楚了「我在模擬什麼」
- [ ] 方法段提到了模型假設和實驗設計
- [ ] 每張圖有圖說（說的是「發現」，不只是「這是什麼圖」）
- [ ] 結論段直接回答研究問題
- [ ] 全文 1–2 頁，圖文各半
