## 核心概念

### 模擬是什麼？

**模擬（Simulation）** 是這樣一件事：

1. 把真實世界的現象**簡化**成一組規則（這叫「模型」）
2. 用電腦**重複執行**這些規則很多次
3. 從大量的結果中**找出規律**

> 比喻：天氣預報員不是靠感覺說「明天會下雨」。他們有一個天氣模型，跑了幾百次模擬，如果 70% 的模擬結果都顯示下雨，才說「降雨機率 70%」。

### 為什麼要重複很多次？

一次實驗告訴你的資訊很有限。

| 實驗次數 | 你能得到的結論 |
|----------|----------------|
| 1 次 | 幾乎什麼都不能說 |
| 10 次 | 非常不穩定，可能結果偏差很大 |
| 100 次 | 開始看出趨勢 |
| 1000 次+ | 結果趨近於理論值 |

### 模型、模擬、結果的關係

```
現實問題
   ↓ 假設 + 簡化
模型（規則）
   ↓ 電腦重複執行
模擬結果
   ↓ 統計 + 視覺化
發現與結論
```

## 今天的範例：骰子模擬

### 完整程式碼

```python
import random
import matplotlib.pyplot as plt

# 函數：模擬一次擲骰子
def roll_dice():
    return random.randint(1, 6)

# 主程式：重複 N 次
N = 1000
results = []

for i in range(N):
    results.append(roll_dice())

# 視覺化：直方圖
plt.figure(figsize=(8, 5))
plt.hist(results, bins=6, range=(0.5, 6.5), edgecolor='black')
plt.xlabel("骰子點數")
plt.ylabel("出現次數")
plt.title(f"擲骰子 {N} 次的結果分佈")
plt.xticks([1, 2, 3, 4, 5, 6])
plt.show()
```

### 程式碼逐行說明

| 程式碼 | 說明 |
|--------|------|
| `import random` | 載入 Python 的隨機數工具 |
| `random.randint(1, 6)` | 產生 1 到 6 之間的隨機整數（包含 1 和 6） |
| `for i in range(N)` | 重複 N 次 |
| `results.append(...)` | 把每次結果加入列表 |
| `plt.hist(...)` | 畫直方圖 |
| `bins=6, range=(0.5, 6.5)` | 把圖分成 6 格，每格對應一個點數 |

## 本週關鍵詞

| 詞彙 | 意思 |
|------|------|
| 模擬（Simulation） | 用電腦重複執行模型來探索結果 |
| 模型（Model） | 對現實的簡化描述（包含假設） |
| 隨機（Random） | 有機率分佈的不確定性，不是「亂來」 |
| 重複試驗 | 跑多次實驗來獲得穩定的統計結果 |
| 直方圖（Histogram） | 顯示數值分佈的長條圖 |
