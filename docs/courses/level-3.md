# Level 3：數值方法與優化

> **NumPy、SciPy、最佳化——從「可以跑」到「跑得有效率又正確」**

---

## 課程目標

完成 Level 3 的學生能夠：

- 使用 NumPy 進行陣列運算與向量化
- 使用 SciPy 求解微分方程（ODE）與最佳化問題
- 設計高級專題，結合數值方法與真實問題
- 進行完整的敏感度分析與參數最佳化

---

## 適合對象

- 完成 Level 2 的學生
- 對數學建模有興趣，想深入 scientific computing
- 考慮申請理工科大學，需要更強的技術作品集

## 10 週課程大綱

| 週次 | 主題 | 核心技能 |
|------|------|----------|
| 第 1 週 | 目標與舊專題回顧 | 找出可以升級的方向 |
| 第 2 週 | NumPy：陣列與向量化 | `ndarray`、broadcasting、效能 |
| 第 3 週 | SciPy：基本數值方法 | 數值積分、根求解 |
| 第 4 週 | SciPy：簡單 ODE 模型 | `solve_ivp`、SIR 進階版 |
| 第 5 週 | 最佳化入門 | `scipy.optimize`、目標函數設計 |
| 第 6 週 | 高級專題選題與方法設計 | 定義問題的數學結構 |
| 第 7 週 | 專題實作：模型與數值方法 | 核心程式完成 |
| 第 8 週 | 專題實作：最佳化與敏感度分析 | 結果分析 |
| 第 9 週 | 技術寫作與簡報 | 報告草稿 |
| 第 10 週 | 作品集展示與整理 | 完整作品集繳交 |
