# 訊息框架

## 一句話定位

> 從「會寫 Python」到「能做大學生等級研究」的完整 Simulation & Optimization Pathway。

## 三大賣點

1. **完整路徑，有看得見的終點**：四個明確階段，每一級都有作品、報告或研究稿，不是學完就沒下文
2. **個別指導，做自己的專題**：每週 1 小時 1-on-1，題目是你的、研究問題是你的，不是抄模板
3. **直接對接申請與課綱**：AP / IB / A Level / 大學申請 / STEM 競賽都能直接用成果，不是為了學習而學習

## 對不同受眾的語氣

### 對家長
- 語氣：清楚、可信、具體
- 強調：孩子會有什麼具體產出，能用在哪裡（申請、課綱、競賽）
- 避免：過度學術術語、誇大承諾

**關鍵句型**：
- 「孩子在這裡會做出一份屬於自己的研究專題，不是複製老師的範例。」
- 「每週都有導師 1 對 1，真正針對孩子的題目給意見，不是群組問答。」
- 「Level 2 的成果可以直接對應 IB IA / AP Research 的評分規準。」

### 對學生
- 語氣：有挑戰感、直接、不浮誇
- 強調：你會學到的技能、做出的東西
- 避免：說教感、過度保證

**關鍵句型**：
- 「不只寫 Python，而是用 Python 解真實的問題。」
- 「你會做出一個自己想得出來、說得清楚的專題。」

### 對補習班合作方
- 語氣：商業導向、強調結構與可操作性
- 強調：留住學生、補齊產品線、收入好算
- 避免：太學術、太抽象

**關鍵句型**：
- 「接在 Python 班後面的自然下一步，學生不用去外面找專題營隊。」
- 「四個級別，每一級有明確產出，家長看得到孩子在進步。」
- 「分潤結構清楚，招生你們負責，我負責教學品質。」

### 對學校合作方
- 語氣：專業、有結構感
- 強調：課綱對接、學術誠信、可驗收的產出

**關鍵句型**：
- 「Level 2–4 可以直接對應 AP Research、IB EE/IA、Honors Project 的要求。」
- 「課程強調研究流程與學術誠信，學生的報告是他們自己做的。」

## 與競爭對手的差異說明

| 比較對象 | 他們做什麼 | 我們做什麼 |
|----------|-----------|-----------|
| APCS / 語法補習班 | 教語法、刷題、備考 | 教建模、模擬、做研究 |
| 一般線上 Python 課 | 跟著老師打 code | 自己做一個完整專題 |
| 專題營隊（暑期） | 2–4 週密集，成果是模板 | 10 週 + 1-on-1，成果是自己的 |
| 升學顧問代寫 | 幫學生寫 / 做 | 教學生自己做，但老師陪你做 |
