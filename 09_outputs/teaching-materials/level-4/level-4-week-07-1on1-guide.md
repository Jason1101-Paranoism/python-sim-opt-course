# Level 4 週 07 — 1-on-1 討論指南

> **本週 1-on-1 的核心任務**：確認實驗設計的嚴謹性，協助學生完成可用的圖表，檢視結果段落是否有效引用圖表且語言符合研究標準。

---

## 本週個別討論聚焦

1. 方法章初稿回饋：老師給書面回饋，1-on-1 口頭討論 1–2 個主要修改點
2. 實驗設計確認：重複次數、控制變因、邊界情境是否都涵蓋
3. 圖表品質檢視：逐張看圖，確認標題/軸標籤/單位/誤差棒
4. 結果段落初稿引導：幫助學生從口語描述轉換為學術語言

---

## 建議討論流程（60 分鐘）

| 時段 | 時間 | 內容 |
|------|------|------|
| 回顧 | 10 分鐘 | 學生報告本週實驗進度，展示初步圖表 |
| 主要討論 | 35 分鐘 | 實驗設計確認 + 圖表逐張檢視 + 結果段落寫作輔導 |
| 下週目標確認 | 10 分鐘 | 確認結果章完整版的完成計劃 |
| 策略備註 | 5 分鐘 | 時程、競賽/申請對圖表格式的特殊要求 |

---

## 具體討論問題清單

### 回顧段（10 分鐘）

1. 「你本週跑了哪些實驗？大概跑了多少次？結果穩定嗎？」
2. 「你的第一張圖出來了嗎？把它打開讓我看看。」
3. 「上週方法章，你有根據我的回饋做任何修改嗎？」

---

### 主要討論（35 分鐘）

**Part 1：實驗設計確認（10 分鐘）**

- 「你改變的參數是 [X]，其他條件你確定都固定住了嗎？」
  - 若學生不確定：「把你的程式打開，我們找一下哪些值是寫死的（hardcoded），哪些是你在改的。」
- 「你每個情境跑了幾次？你怎麼確定次數夠了？」
  - 若學生沒有做收斂測試：「我們現在就做一個快速測試——跑 10 次、100 次、500 次，看看平均值有沒有收斂。」
- 「你有測試邊界情境嗎？最小值和最大值的結果是什麼？」

**Part 2：圖表逐張檢視（15 分鐘）**

請學生打開每張圖，依序確認：

| 檢查項目 | 是/否 | 若否，如何修改 |
|----------|-------|--------------|
| 橫軸有標籤和單位 | | |
| 縱軸有標籤和單位 | | |
| 有圖例（若多條線） | | |
| 有誤差棒（若隨機模擬） | | |
| 字體 ≥ 12pt | | |
| 線條粗細 ≥ 2 | | |

- 「這張圖的指標（[指標名]）能回答你的 RQ 嗎？怎麼說？」
- 「如果你只能給讀者看一張圖，你會選哪一張？為什麼？」

**Part 3：結果段落引導（10 分鐘）**

若學生已有初稿：逐句讀，用以下問題引導修改：
- 「這句話是在說事實還是在解釋原因？如果是解釋，我們把它搬到討論章。」
- 「這裡你說『比較多』，多多少？能加具體數字嗎？」
- 「圖 1 在你的文字裡有被引用嗎？在哪一句？」

若學生尚無初稿：帶學生口頭描述，老師幫助整理成學術語句：
- 「你的圖 1 在說什麼？用一句話告訴我。」（老師記錄口頭說法）
- 「好，現在把『我發現』改成『如圖 1 所示』，把『比較多』改成具體數字，再念一遍。」

---

### 下週目標確認（10 分鐘）

請學生確認以下週 08 前的目標：

| 項目 | 學生確認 |
|------|---------|
| 圖表數量 | 目標完成 [N] 張 |
| 結果段落字數 | 目標完成 [X]–[Y] 字 |
| 是否需要補做任何情境 | □ 是：[描述] / □ 否 |
| 方法章修改完成時間 | |

**確認下週目標（具體、可驗收）**：
- 週 08 上課前：完整結果章初稿（含所有圖表 + 描述段落）
- 週 08 上課前：方法章依老師回饋修改完畢（若有修改）

---

### 策略備註（5 分鐘）

**若學生目標是競賽投稿**：
- 「這個競賽對圖表格式有特殊要求嗎？（解析度、顏色、向量格式？）」
- 「你現在的圖表語言（中/英文）和投稿要求一致嗎？」

**若學生目標是大學申請 writing sample**：
- 「你有沒有確認目標學校/申請系統接受什麼格式（PDF？Word？）」
- 「你的圖表在轉 PDF 後解析度還可以嗎？要確認一下。」

**若學生整體進度落後**：
- 評估：是實驗設計問題、程式問題，還是時間管理問題？
- 「我們需要討論一下，哪些實驗是必做的，哪些可以捨去。你的 RQ 最核心的那個問題需要哪一張圖來回答？其他的先放。」

---

## 本週學生應達成的里程碑

| 里程碑 | 說明 |
|--------|------|
| 主要實驗完成 | 至少一組系統性參數掃描，重複次數達標 |
| 第一張關鍵圖完成 | 研究等級，有標題/軸標籤/單位/誤差棒 |
| 結果段落初稿存在 | 即使只有 2–3 句，有引用圖/表，無解釋性語句 |
| 方法章修改啟動 | 能說出上週方法章 1–2 個主要改進方向 |
