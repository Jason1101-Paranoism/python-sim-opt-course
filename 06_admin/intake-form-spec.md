# 報名表欄位規格

> 本文件定義課程報名表（Google Form）的欄位結構、選項與填寫說明。
> 實作時以 Google Form 為主，補習班行政系統為輔。

---

## 表單用途

- **學生入班前的基本資料收集**
- 判斷是否需要安排程度測驗（直接入班高級 Level 者）
- 確認時間偏好、優惠資格與緊急聯絡資訊
- 讓老師在第一次 1-on-1 前有足夠背景

---

## 第一節：報名課程資訊

| 欄位 | 類型 | 必填 | 選項 / 說明 |
|------|------|------|------------|
| 報名 Level | 單選 | ✅ | Level 1 / Level 2 / Level 3 / Level 4 |
| 是否為直接入班申請（跳過 Level 1）| 單選 | ✅ | 是，我有 Python 基礎想直接申請較高 Level / 否，從 Level 1 開始 |
| 偏好梯次 | 單選 | ✅ | 暑期班（7–9 月）/ 學期班 A（9–12 月）/ 學期班 B（2–6 月）/ 待安排 |
| 偏好上課時段 | 多選 | ✅ | 平日下午（17:00–19:00）/ 平日晚間（19:00–21:00）/ 週六上午 / 週六下午 / 週日上午 / 週日下午 |
| 有無優惠資格 | 多選 | ✅ | 早鳥報名 / 三人以上團報 / 同時報名 2 級以上 / 補習班舊生 / 特約學生 / 無 |

---

## 第二節：學生基本資料

| 欄位 | 類型 | 必填 | 說明 |
|------|------|------|------|
| 學生姓名 | 短文字 | ✅ | 填寫真實姓名（課程管理用） |
| 學生英文姓名 | 短文字 | 選填 | 若有國際文件需求（結業證書、報告封面） |
| 就讀學校 | 短文字 | ✅ | |
| 年級 | 單選 | ✅ | 國中 7 年級 / 8 年級 / 9 年級 / 高中 10 年級 / 11 年級 / 12 年級 / 其他 |
| 就讀類型 | 單選 | ✅ | 一般公立學校 / 私立學校 / 國際學校 / 雙語學校 / 實驗學校 / 自學 / 其他 |
| 是否修讀 AP / IB / A-Level | 單選 | 選填 | 是（請說明課程）/ 否 |

---

## 第三節：Python 程度自評

| 欄位 | 類型 | 必填 | 說明 |
|------|------|------|------|
| 學習 Python 的經歷 | 單選 | ✅ | 從未寫過 Python / 上過入門課（會變數、迴圈、函數）/ APCS 準備中或已通過 / 有自己做過小專案 / 其他 |
| 可以在自己電腦安裝並執行 Python | 單選 | ✅ | 是 / 否，需要協助 |
| 曾參加過的相關課程或活動 | 長文字 | 選填 | 如：補習班 Python 班、程式競賽、專題營隊等，簡短說明即可 |
| 有無特別感興趣的題目領域 | 長文字 | 選填 | 如：物理、生態、金融、交通、遊戲等；無想法也可空白 |

| 欄位 | 類型 | 必填 | 說明 |
|------|------|------|------|
| 家長 / 監護人姓名 | 短文字 | ✅ | |
| 與學生關係 | 單選 | ✅ | 父 / 母 / 監護人 / 其他 |
| 聯絡電話 | 短文字 | ✅ | 緊急聯絡用 |

| LINE ID（選填） | 短文字 | 選填 | 補習班慣用溝通工具 |
| 主要溝通偏好 | 單選 | ✅ | LINE / Email / 電話 |

---

## 第五節：其他資訊

| 欄位 | 類型 | 必填 | 說明 |
|------|------|------|------|
| 如何得知本課程 | 單選 | ✅ | 補習班直接介紹 / 朋友推薦 / 社群媒體 / 學校 / 其他 |
| 有無任何特殊學習需求或身體狀況需要告知 | 長文字 | 選填 | 僅作為教學調整參考，不影響報名資格 |
| 對課程有什麼期待或問題 | 長文字 | 選填 | 供老師在第一次 1-on-1 前參考 |

## 直接入班申請者的附加欄位

> 僅顯示給第一節勾選「是，直接申請較高 Level」者。

| 欄位 | 類型 | 必填 | 說明 |
|------|------|------|------|
| 目標 Level | 單選 | ✅ | Level 2 / Level 3 / Level 4 |
| 說明你的 Python 使用經驗 | 長文字 | ✅ | 做過什麼、用過什麼工具、完成什麼作品，約 100–200 字 |
| 方便接受程度測驗的時段 | 短文字 | ✅ | 填寫 2–3 個可用時段（老師安排 30 分鐘線上測驗） |

---

## 表單設計注意事項

- 第一節「是否直接入班」應使用條件顯示（Conditional Logic），只對直接入班者顯示附加欄位
- 所有資料依 `consent-and-ip-policy.md` 說明使用方式，表單結尾需附隱私聲明
- 送出後自動回覆確認信，說明下一步（測驗安排 or 等待開班通知）
- 表單回覆匯出至 Google Sheets，補習班行政與老師共用存取

---

## 報名完成後的標準動作

1. 自動確認信發出（含下一步說明）
2. 老師於 **2 個工作天內**確認報名資料
3. 直接入班者：安排程度測驗（見 `placement-test-spec.md`）
4. 一般入班者：等待開班通知，確認後寄送 onboarding 清單（見 `onboarding-checklist.md`）
