# Python Simulation & Optimization Pathway
## 補習班合作提案

---

## 第一部分：這個系列是什麼

**一句話說清楚：**
這是一個接在 Python / APCS 班後面的「進階研究 Pathway」，帶學生從「會寫語法」走到「能做出一份研究報告」。

**課程結構（四個 Level，各 10 週）：**

| Level | 名稱 | 產出 |
|-------|------|------|
| 1 | 模擬入門 | 個人模擬程式 + 圖文報告 |
| 2 | AP/IB/A-Level 專題 | 完整研究型報告 |
| 3 | 數值方法與優化 | 技術報告 + 作品集 |
| 4 | Research / Capstone | 10–15 頁研究報告 + 模擬口試 |

每一級都有明確的產出，家長看得到孩子在進步。

---

## 第二部分：你們的學生在哪個痛點

你們的 Python 班學生，上完語法課之後，下一步在哪裡？

常見狀況：
- 學完語法、考完 APCS，家長問「然後呢？」
- 學生想做專題，但補習班沒有對應的進階課程
- 學生去外面找「專題輔導班」或「暑期營隊」——你們的客戶流失了

**這個系列就是要接住這群學生。**

讓他們在你們的體系裡繼續升課，不用去外面找。

---

## 第三部分：這系列如何解決

**留住學生的完整路徑：**

```
你們的 Python 基礎班
        ↓
Level 1：模擬入門（10 週）
        ↓
Level 2：AP/IB/A-Level 專題（10 週）
        ↓
Level 3：數值方法與優化（10 週）
        ↓
Level 4：Research / Capstone（10 週）
```

一個學生從 Level 1 到 Level 4 的總課程費：**124,000 元**（定價全額）

即使只留住一個學生完成 Level 1–2，也是 60,000 元的課程收入。

---

## 第四部分：課程結構細節

每一個 Level：

| 項目 | 規格 |
|------|------|
| 課程長度 | 10 週 |
| 每週小班課 | 2 小時（4–8 人） |
| 每週個別指導 | 每位學生 1 小時 1-on-1 |
| 每生總時數 | 30 小時 |
| 建議班級人數 | 4–8 人 |

**為什麼有 1-on-1？**
這是本系列最大的差異化。每週 1 小時一對一，讓學生做自己的題目，而不是複製模板。這種深度輔導，家長在同等價位的市場上很難找到替代品。

## 第五部分：定價與分潤

### 定價（對外標價）

| Level | 定價 |
|-------|------|
| Level 1–3 | 30,000 元 / 級 |
| Level 4 | 34,000 元 / 級 |

優惠規則跟你們現有班級一致（早鳥 88 折、團報 88 折、特約 9 折、續班 9 折），家長看同一張 DM 會覺得統一，好溝通。

### 分潤方案（擇一）

**方案 A：你們負責招生 + 行政 + 場地**
- 你（老師）：60%
- 補習班：40%

**方案 B：老師也帶學生進來，你們負責場地與行政**
- 你（老師）：70%
- 補習班：30%

### 試算（以 Level 1–3，定價 30,000 元）

**6 人班（無折扣）：**

| | 方案 A（6:4） | 方案 B（7:3） |
|--|--------------|--------------|
| 總營收 | 180,000 | 180,000 |
| 老師實拿 | 108,000 | 126,000 |
| 補習班實拿 | 72,000 | 54,000 |

**4 人班（無折扣）：**

| | 方案 A | 方案 B |
|--|--------|--------|
| 總營收 | 120,000 | 120,000 |
| 老師實拿 | 72,000 | 84,000 |
| 補習班實拿 | 48,000 | 36,000 |

**早鳥 88 折，6 人班：**

| | 方案 A | 方案 B |
|--|--------|--------|
| 總營收（26,400×6） | 158,400 | 158,400 |

## 第七部分：合作方 FAQ

**Q：萬一招不滿怎麼辦？**
低於 3 人不開班，課程往後順延。我們不會為了開班而讓品質打折。

**Q：我們要怎麼說明這是什麼課？**
你們的學生完成 Python 基礎後，這就是「下一步」。課程有完整的招生 DM 和一頁紙總覽，你們可以直接使用，不需要額外設計說明材料。

**Q：上課需要什麼設備？**
學生自帶筆電，安裝 Python（Anaconda 或 VS Code 均可）。教室只需要有投影或大螢幕顯示器。

**Q：學生跑掉了怎麼辦？**
每一級都有固定的退費規則（依上課週數比例退費），補習班不需要另設退費政策，沿用本課程標準。

**Q：我想先開 Level 1 試試看可以嗎？**
可以。先開 Level 1，跑完一期評估效果後再決定是否繼續 Level 2 以上，沒有強制捆綁。
