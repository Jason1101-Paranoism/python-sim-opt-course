# Python Simulation & Optimization Pathway

> **從模擬入門到研究級專題的完整學習路徑**

---

## 這是什麼課？

一套針對高中生設計的 **Python 模擬與優化** 四級課程。學生不只學程式語法，而是學會用 Python **建模、模擬、分析資料、寫研究報告**。

每一級產出一份真實作品：可以放進學習歷程、AP/IB 報告、甚至大學申請作品集。

---

## 四級學習路徑

| 級別 | 課程名稱 | 適合對象 | 核心產出 |
|------|----------|----------|----------|
| Level 1 | 模擬入門 | 高一或 Python 初學者 | 隨機模擬小專題 + mini report |
| Level 2 | AP/IB 專題 | 需要 IA/EE/side project 的學生 | 完整 project 報告 + 簡報 |
| Level 3 | 數值方法與優化 | 想深入 NumPy/SciPy 的學生 | 進階數值模型 + 優化實驗 |
| Level 4 | Research Capstone | 有意申請研究型大學或競賽的學生 | 研究論文等級報告 + 口試模擬 |

---

## 為什麼選這套課？

=== "不只是語法課"
    課程核心是**建模思維**，不是背 Python 語法。學生每週都有可以展示的東西。

=== "真實作品產出"
    每一級都有期末成果：report、slide、code。可直接用於學習歷程與大學申請。

=== "個別指導"
    小班 4–8 人 + 每人每週 1 小時個別指導。老師真的了解每位學生的狀況。

=== "系統化路徑"
    四級環環相扣，從「能跑出模擬結果」到「能寫研究報告」，每一步都有明確里程碑。
