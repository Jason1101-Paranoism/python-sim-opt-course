# Level 1 週 2：作業單（Assignment Brief）

---

## 本週作業：彩票 or 生日悖論模擬

**截止時間**：下次上課前 1 小時
**預計完成時間**：60–90 分鐘

---

## 選一個做（彩票 或 生日悖論）

### 選項 A：彩票模擬

**要繳交的東西**：
- [ ] Python 程式檔
- [ ] 執行結果截圖（包含模擬中獎率 vs 理論中獎率的對比）
- [ ] 說明欄（見下方）

**具體要求**：
1. 程式能正確模擬大樂透（6 個不重複號碼，範圍 1–49）
2. 跑至少 **10,000 次**模擬，輸出模擬中獎率
3. 在說明欄回答：「如果每週買一張，平均要買幾年才中頭獎？」（可以計算，也可以程式算）

### 選項 B：生日悖論

**要繳交的東西**：
- [ ] Python 程式檔
- [ ] 折線圖截圖（x 軸：人數，y 軸：配對機率）
- [ ] 說明欄（見下方）

**具體要求**：
1. 程式能計算 2–60 人各自的配對機率
2. 折線圖有：標題、x/y 軸標籤、50% 參考線
3. 在說明欄回答：「23 人時的配對機率是多少？這跟你的直覺一樣嗎？」
