# Level 3 週 07 作業單：專題核心程式（含數值方法）

## 作業名稱與目標

**作業名稱**：高級專題核心程式 v1.0

**目標**：
完成你的專題程式的核心數值計算部分，包含：至少一個 SciPy 數值方法（ODE solver 或數值積分）的整合、基礎的結果合理性驗證、以及至少一張圖表輸出。

---

## 繳交清單

| 項目 | 格式 | 說明 |
|------|------|------|
| 主程式 | `.py` 檔 | 必須可以直接執行（不能有未解決的 ImportError 或 NameError） |
| 結果圖表 | `.png` 或 `.pdf` | 至少一張，有標題、座標軸標籤 |
| 收斂測試結果 | 程式輸出截圖或附在 `.py` 的 print 輸出 | 至少測試 3 種步長或精度設定 |
| 邊界案例記錄 | `.py` 中以注釋說明，或獨立文字段落 | 至少 2 個邊界案例 |

**截止時間**：下次 group 課前（週 08 上課前 24 小時）

---

## 具體要求

1. **程式必須可以執行**：老師會直接執行你的 `.py` 檔，若有未解決的錯誤視同未繳
2. **必須使用 SciPy**：核心計算至少用一個 SciPy 函數（`solve_ivp`、`quad`、`dblquad` 等），不可以只用 for 迴圈
3. **程式需有中文注釋**：每個主要段落（模型定義、參數設定、求解、畫圖）需有一行以上的中文說明
4. **收斂測試需有輸出**：執行程式時，終端機需要顯示不同步長的對照結果（`print` 即可）
5. **圖表需有標籤**：`xlabel`、`ylabel`、`title` 至少要有，`legend` 視情況加

---

## 自我檢查清單

繳交前，請確認以下每一項：

- [ ] 程式從頭到尾可以執行，沒有錯誤
- [ ] 有用到 `scipy.integrate.solve_ivp` 或其他 SciPy 數值函數
- [ ] 程式中有中文注釋，說明每個段落在做什麼
- [ ] 收斂測試：測試了至少 3 種不同步長，並 print 出對照結果
- [ ] 邊界案例：設計了至少 2 個極端條件測試，並說明預期結果與實際結果
- [ ] 圖表有標題、X 軸、Y 軸標籤
- [ ] 圖表儲存為 `.png`，一起繳交
- [ ] 程式碼結構清楚：模型定義、參數設定、求解、畫圖 各成一段

---

## 常見失分項目

| 問題 | 說明 |
|------|------|
| 程式無法執行 | 繳交前請在乾淨環境重新跑一次 |
| 只用 for 迴圈，沒有用 SciPy | 這週的學習目標就是整合 SciPy，必須達到 |
| 圖表沒有標籤 | 「這是什麼的圖」如果看不出來，圖表沒有溝通價值 |
| 收斂測試只跑一個步長 | 收斂測試需要「比較」不同設定，才能看出趨勢 |
| 邊界案例只有敘述，沒有程式執行結果 | 需要實際跑出來，不是只說「我猜應該是…」 |
