# Level 1 週 1：作業單（Assignment Brief）

---

## 本週作業：骰子模擬程式

**截止時間**：下次上課前 1 小時

**預計完成時間**：60–90 分鐘

**目標**：完成一個「擲骰子 N 次並畫直方圖」的程式，並截圖繳交。

---

## 要繳交的東西

- [ ] Python 程式檔（`.py` 或 `.ipynb`）
- [ ] 直方圖截圖（至少一張，N=1000）
- [ ] 簡短說明（3–5 句話，純文字貼在 Google Drive 或寫在程式注解裡）

**命名規則**：`level1-week01-你的名字縮寫.py`
例：`level1-week01-LL.py`

**上傳位置**：老師的 Google Drive 共用資料夾 → 你的個人子資料夾

---

## 作業要求

### 必須完成（基本版）

1. **程式可以執行**，不能有 import error 或 syntax error
2. **直方圖要有**：
   - [ ] 標題（包含 N 的值）
   - [ ] x 軸標籤（骰子點數）
   - [ ] y 軸標籤（出現次數）
3. **嘗試至少兩個不同的 N 值**，在說明中寫下你觀察到的差異

### 選做（加分題）

4. 把骰子換成 12 面骰，程式能正確執行並輸出合理的圖
5. 在同一張畫布上畫出 N=100 和 N=10000 的對比圖

---

## 說明欄（必填，貼在程式注解或 Google Doc）

請用自己的話回答這三個問題（每個 1–2 句話）：

**1. 這個程式在模擬什麼？**

_______________________________________________

**2. N 的大小對結果有什麼影響？**

_______________________________________________

**3. 你有遇到什麼卡住的地方嗎？**（沒有也要寫「沒有」）

_______________________________________________

---

## 自我檢查清單

繳交前確認：

- [ ] 程式可以執行，沒有 error
- [ ] 直方圖有標題、x 軸標籤、y 軸標籤
- [ ] 圖看起來合理（每個點數出現次數大致相近）
- [ ] 檔案命名正確
- [ ] 說明欄有填寫

---

## 還在卡關？

**先試試這些**：

1. 把程式一段一段分開執行，確認每段都能跑
2. 把 `plt.show()` 改成 `plt.savefig("test.png")`，看是否能存圖
3. 把問題截圖，帶到 1-on-1 討論

**不要做的事**：

- 不要因為有一個 error 就整個刪掉重來
- 不要等到上課前一小時才開始
