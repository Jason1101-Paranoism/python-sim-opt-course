# 結業證書規格

> 本文件定義課程結業證書的欄位、核發條件與版面設計規範。
> 主要用途：學生申請檔案、作品集附件、學校說明用。

---

## 核發條件

下列條件**全部**達成，方核發結業證書：

| 條件 | 說明 |
|------|------|
| 出席率 | Group 課出席 ≥ **7 週**（含錄影補看，共 10 週） |
| 1-on-1 完成 | 完成 ≥ **8 次** 1-on-1（共 10 次） |
| 期末成果繳交 | 期末報告或專題成果已繳交（不論評等） |
| 期末展示 | 參與期末展示（若因不可抗力缺席，需與老師另行說明） |

> 未達核發條件者，可由老師視情況出具**修課證明**（載明已修週次），不出具結業證書。

---

## 中文版欄位

| 欄位 | 內容說明 |
|------|---------|
| 標題 | 結業證書 |
| 學生姓名 | 中文姓名（依報名表） |
| 課程名稱 | Python Simulation & Optimization Pathway — Level [X]：[課程副標題] |
| 課程副標題 | Level 1：模擬入門 / Level 2：AP/IB/A-Level 專題 / Level 3：數值方法與優化 / Level 4：Research Capstone |
| 課程時數 | 共 30 小時（含 group 課 20 小時 + 個別指導 10 小時） |
| 課程期間 | YYYY 年 MM 月 DD 日 至 YYYY 年 MM 月 DD 日 |
| 期末成果 | [一句描述學生的專題題目與成果，例如：「完成交通流量模擬專題報告」] |
| 核發機構 | [課程方名稱] |
| 核發日期 | YYYY 年 MM 月 DD 日 |
| 簽名欄 | 授課教師簽名（手寫或電子簽名） |

## 英文版欄位（選用，Level 3–4 或國際申請用）

| 欄位 | 內容說明 |
|------|---------|
| 標題 | Certificate of Completion |
| 學生姓名 | English name（依報名表，無則用中文拼音） |
| 課程名稱 | Python Simulation & Optimization Pathway — Level [X]: [English subtitle] |
| 英文副標題 | Level 1: Introduction to Simulation / Level 2: AP/IB/A-Level Project / Level 3: Numerical Methods & Optimization / Level 4: Research Capstone |
| 課程時數 | 30 hours (20 hrs group instruction + 10 hrs individual mentoring) |
| 課程期間 | [Month DD, YYYY] to [Month DD, YYYY] |
| 期末成果 | [One sentence describing the student's final project, e.g., "Completed a simulation study on urban traffic flow optimization"] |
| 核發機構 | [Instructor/Organization name] |
| 核發日期 | [Month DD, YYYY] |
| 簽名欄 | Instructor signature |

## 版面設計規範

| 項目 | 規格 |
|------|------|
| 紙張規格 | A4 橫式（Landscape）或直式（Portrait），視設計決定 |
| 檔案格式 | PDF（供列印與電子傳送），原始設計存於 Google Slides 或 Canva |
| 字型 | 中文：Noto Serif TC 或 Source Han Serif；英文：Serif 字體（如 Georgia） |
| 主色調 | 深藍或深灰（正式感），避免過於鮮豔 |
| 必要元素 | 課程 Logo（若有）、邊框裝飾（簡潔）、簽名欄、印章欄（若有） |
| 不得包含 | 過度渲染的設計元素；與課程無關的機構背書 |

---

## 核發流程

1. 第 10 週 group 課結束後，老師確認學生是否達到核發條件
2. 填入學生資料與期末成果描述
3. 出具 PDF（中文版 / 英文版 / 中英雙版）
4. 連同期末報告一起傳送給學生與家長
5. 老師留存一份副本（Google Drive），保存至課程結束後 1 年

---

## 修課證明（未達結業條件者）

若學生未達結業條件但確有修課（如因病缺席多次），可出具修課證明，格式較簡單：

| 欄位 | 內容 |
|------|------|
| 標題 | 修課證明 |
| 學生姓名 | 同上 |
| 修課課程 | 同上 |
| 實際出席週數 | X 週（共 10 週） |
| 備註 | [說明未完整修畢的原因，如：因健康因素，已修 6 週課程] |
| 核發日期 | 同上 |

> 修課證明不等同於結業，不建議用於正式申請，但可作為說明用途。
