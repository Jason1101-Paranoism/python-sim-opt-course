# 合作模式

## 線下新竹版（補習班 Pathway）

**主體**：小班課 + 1-on-1，接在 APCS／語法班後面的「進階路徑」

**定位**：幫本地學生從會寫 Python 走到會做專題與研究；幫補習班留住原本要流出去做專題營隊的學生

**合作對象**：
- 在地補習班（主要合作對象）
- 實驗學校 / 雙語學校：作為正式選修課、after-school program、暑期營

**收費結構**：TWD 為主，走補習班報名系統

---

## 國際線上 Premium 一對一版

**主體**：高度客製的 research mentorship / capstone 專案

**定位**：針對申請美國／英國 top schools 的 AP/IB/國際高中生，一對一帶完一個「可寫進 SOP 和 CV」的模擬 + 最佳化研究專題

**合作通路**：
- Upwork / Preply / Superprof / AmazingTalker
- LinkedIn 直接接觸
- 升學顧問機構串接

**收費結構**：USD 為主，project fee 制

---

## 三個 Premium 產品層級（國際線）

### 層級 1：高中 AP/IB/A-Level「Side Project / IA / AP Research」版
- **對象**：AP Research、IB IA、A Level、想做 serious side project 的高中生
- **內容**：10–12 週，每週 1h live + 1h async feedback，產出專題報告（對齊 rubric）
- **建議定價**：2,400–3,000 USD / 10–12 週（project fee，不以 hourly 報價）

### 層級 2：本科/碩士「論文建模與模擬顧問」版
- **對象**：在 Stanford、MIT 等校念 STEM / CS / Robotics / Quant 的學生，卡在「數學理論如何變成 code」
- **內容**：協助從問題到模型、數值方法選擇、Python 實作、結果圖與誤差分析（不代寫論文）
- **建議定價**：250–350 USD/hr；或 3,000–6,000 USD/project

### 層級 3：業界工程 / Quant 轉職「架構 + 數學建模」版
- **對象**：SWE 轉 Quant / AI / Robotics，或架構亂、數學建模弱的工程師
- **內容**：架構 refactor、模組化、測試設計；SDE、optimal control、portfolio optimization mini project
- **建議定價**：300–400 USD/hr；轉職 package 5,000–10,000 USD

## 一套課雙軌使用

四個 Level 的內容不變，本質是一套 curriculum engine：

- **線下**：跑小班版 + 固定價（NTD 為主）
- **線上**：抽出 Level 2 + Level 4 精華，包裝成 2–3 種一對一套餐（USD 為主）

同一份 CLAUDE.md + md 規格：
- 生成中文教材 + 行政文件 → 線下用
- 生成英文版課程頁 / Upwork profile / proposal template → 國際學生用
