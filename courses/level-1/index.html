    Level 1：模擬入門

    用 Python 跑真實的隨機模擬，第一次就產出有意義的結果

    課程目標

    完成 Level 1 的學生能夠：

    • 用 Python 建立隨機模擬模型（骰子、隊伍、疫情擴散）
    • 用圖表呈現模擬結果並說明意義
    • 獨立完成一份 mini project，包含程式碼 + 分析說明
    • 養成「先建模、再跑模擬、再分析」的思考習慣

    適合對象

    • 已有基礎 Python 能力（會迴圈、函數、list），想把程式用在真實問題上
    • 對數學或科學有興趣，但不確定要做什麼專題
    • 想為 AP/IB/學習歷程準備，需要一個起點
    • APCS 考完、想找「下一步」的學生

    先修要求：Python 基礎（變數、迴圈、函數）。完全零基礎者建議先完成 Python 入門課程。

    10 週課程大綱

    週次 主題 核心技能
    第 1 週 什麼是模擬？骰子實驗 random、基本迴圈、matplotlib
    第 2 週 機率遊戲：彩票與生日悖論 大數法則、直方圖
    第 3 週 排隊系統：等待時間模擬 多變數模型、參數調整
    第 4 週 成長與衰退：差分模型 SIR 模型、指數成長、折線圖
    第 5 週 小專題 1：個人 mini project 獨立建模、問題定義
    第 6 週 視覺化：用圖說故事 seaborn、多圖排版、圖表標題
    第 7 週 參數與敏感度 參數掃描、敏感度分析
    第 8 週 小組專題工作坊 協作分工、版本管理
    第 9 週 小報告寫作練習 結果詮釋、圖文整合
    第 10 週 展示與反思 口頭說明、Q&A 應答

    期末成果

    每位學生完成一份 Mini Project，包含：

    • Python 模擬程式碼（可執行）
    • 1–2 頁分析說明（問題 → 模型 → 結果 → 結論）
    • 2–3 張圖表（清楚標示軸標題與解釋）
