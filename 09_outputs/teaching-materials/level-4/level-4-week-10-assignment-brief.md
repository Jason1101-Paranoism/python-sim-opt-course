# Level 4 週 10 作業單 — 期末展示準備

## 作業名稱

**期末研究展示：研究簡報 + 口試準備 + 研究報告最終版**

## 作業目標

本週作業有兩部分：
1. **課前**：完成期末簡報投影片，並至少在家練習計時一次
2. **課後**：依據期末展示的口試回饋，修改研究報告（若有需要），整理成可用於競賽或大學申請的版本

---

## 期末展示規格（上課當天）

### 簡報時間規格

- 正式簡報：**8–10 分鐘**
- 問答時間：**5 分鐘**（老師扮演口試委員）

### 投影片規格

| 項目 | 要求 |
|------|------|
| 張數 | 6–9 張（含封面，不含備用投影片） |
| 格式 | PowerPoint / Google Slides / Keynote（任一） |
| 字體大小 | 標題 ≥ 32pt；內文 ≥ 24pt |
| 語言 | 中文或英文（與你的研究報告語言一致） |
| 圖表 | 從報告中直接使用（已有完整標籤），無需重做 |

### 投影片結構要求

| 投影片 | 必要內容 |
|--------|---------|
| 封面 | 研究標題、作者姓名 |
| 背景與問題 | 研究動機、RQ（清楚說出你的研究問題） |
| 文獻背景 | gap（前人沒做到的）+ 本研究的切入點（最多 2–3 篇引用） |
| 研究方法 | 模型、工具、實驗設計（簡化版，可用流程圖） |
| 主要結果（2 張） | 每張一個關鍵發現（圖或表），有標題說明 |
| 討論與意義 | 最重要的 1–2 個討論點，說出意義 |
| 結論 | 一句話回答 RQ + 最重要的限制 + 一個未來方向 |

---

## 要繳交的東西

### 課前繳交（週 10 上課當天前）

1. **期末簡報投影片檔**
   - 上傳至指定位置（Google Drive / 共享連結），或當天攜帶
   - 備份在雲端（避免當天設備問題）

2. **完整研究報告初稿**（若週 09 的版本尚未完整，這週補齊）
   - 所有七個章節都有內容
   - Google Docs 共享連結（老師留最後一輪回饋）

### 課後繳交（展示後 1 週內，若有後續用途）

3. **研究報告修正版**（選擇性，若要投競賽或申請）
   - 依展示回饋修改 2–3 個主要問題
   - 確認所有引用格式正確

---

## 期末展示的評量重點

本次展示**不計分**，但老師會逐項給書面回饋，作為你後續修改報告和準備競賽/申請的參考。

| 評量面向 | 說明 |
|---------|------|
| 研究問題清晰度 | 聽眾能在 1 分鐘內了解你在研究什麼 |
| 方法說明合理性 | 能說清楚做了什麼、為什麼這樣做 |
| 結果呈現 | 圖表能直接支持 RQ，有具體數值 |
| 討論深度 | 說出結果的意義，不只是重複結果 |
| 問答應對 | 能合理回應方法論和結果的問題 |

---

## 口試問答準備清單

展示前，請確認你能回答以下問題：

**關於研究問題**：
- [ ] 我能用一句話說清楚我的 RQ
- [ ] 我能說出前人研究沒有解決的 gap
- [ ] 我能說明為什麼這個 RQ 值得研究

**關於方法**：
- [ ] 我能說明為什麼選擇這個模型（而不是其他模型）
- [ ] 我能說明我的主要假設，以及假設的理由
- [ ] 我能說明為什麼選擇這個重複次數（夠了嗎？）

**關於結果**：
- [ ] 我能解釋每張圖的主要趨勢，以及為什麼出現這個趨勢
- [ ] 我知道我的研究中最令人意外的發現是什麼
- [ ] 我能引用文獻說明我的結果是否與前人一致

**關於限制與未來工作**：
- [ ] 我能說出最重要的一個限制，以及它如何影響我的結論
- [ ] 我能說出一個具體的未來研究方向及其理由

## 期末展示後的後續選項

展示完成後，你有以下幾個選項可以繼續推進你的研究：

### 選項 1：投科展或研究競賽
- 確認目標競賽的截止日期和格式要求
- 老師可在完課後繼續提供回饋（每週一次，最多 2–3 次）
- 常見競賽：學科能力競賽、科技部青年學者計畫（大學申請參考用）、校內科展

### 選項 2：大學申請 Writing Sample
- 確認目標學校/申請系統的格式要求（長度、語言、格式）
- 若需要英文版本，老師可確認學術語言品質
- 建議附上摘要（Abstract，150 字以內）

### 選項 3：保留為個人作品集
- 整理成 PDF，附在你的個人網站或申請材料
- 無截止壓力，可以慢慢修改

## 自我檢查清單（展示前完成）

**投影片**
- [ ] 封面有標題、作者姓名
- [ ] 每張投影片都有「核心訊息句」標題（不是只有章節名稱）
- [ ] 圖表有完整標題和軸標籤
- [ ] 字體 ≥ 24pt
- [ ] 在家計時演練了至少一次，總時長在 8–10 分鐘內

**口試準備**
- [ ] 我能回答「你為什麼要做這個研究」
- [ ] 我能回答「你做了什麼假設，為什麼」
- [ ] 我能回答「你的結論能直接應用嗎，有什麼限制」

**技術準備**
- [ ] 投影片有雲端備份（Google Drive / OneDrive）
- [ ] 確認上課當天可以用來展示的設備（投影或螢幕分享）
