# 課程總覽地圖

## 四級一覽表

| Level | 名稱 | 先修條件 | 每週時數 | 總時數 | 期末產出 |
|-------|------|----------|----------|--------|----------|
| 1 | 模擬入門 | 基礎 Python（會迴圈、函數） | 2h 小班 + 1h 1-on-1 | 30h | 個人小專題程式 + 1–2 頁報告 + 小組展示 |
| 2 | AP/IB/A-Level 專題 | Level 1 完成，或通過程度測驗 | 2h 小班 + 1h 1-on-1 | 30h | 完整專題報告 draft + 程式 + 簡報 |
| 3 | 數值方法與優化 | Level 2 完成，或有完整 side project 基礎 | 2h 小班 + 1h 1-on-1 | 30h | 高級專題：程式 + 報告（方法章） + 作品集 |
| 4 | Research / Capstone | Level 3 完成，或有明確研究題目與程式基礎 | 2h 小班 + 1h 1-on-1 | 30h | 研究報告初稿（10–15 頁）+ 研究簡報 |

## 晉級關係

```
Level 1 → Level 2 → Level 3 → Level 4
  ↑                   ↑
（也可由程度測驗直接進入 Level 2 或 Level 3）
```

詳細晉級規則見 `progression-rules.md`。

## 各級核心目標

### Level 1：模擬入門
- 建立對機率、隨機性、簡單動態系統的直覺
- 學會用 Python 做重複實驗與視覺化
- 產出個人小專題與一頁報告

### Level 2：AP/IB/A-Level 專題
- 完成從研究問題 → 模型 → 模擬 → 分析 → 報告的完整流程
- 產出可直接用於 AP Research、IB IA、A Level coursework 的專題

### Level 3：數值方法與優化
- 導入 NumPy 向量化、SciPy 數值方法、正式最佳化觀念
- 讓作品在技術與數學上接近大學初階課程水平

### Level 4：Research / Capstone
- 學會正式文獻探討與研究寫作
- 完成一份接近大學生 capstone / 本科論文等級的研究型專題

## 1-on-1 聚焦方向

| Level | 1-on-1 重點 |
|-------|------------|
| 1 | 補個人薄弱概念、協助題材選擇與 debug |
| 2 | 題目定稿與修正、模型設計、報告逐段潤飾 |
| 3 | 指導數值方法選擇與調參、打磨技術說服力 |
| 4 | 協助選定與閱讀關鍵文獻、逐段看 lit review / 方法 / 討論、對齊目標大學或競賽要求 |
