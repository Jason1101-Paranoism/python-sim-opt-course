# Level 1 週 5：1-on-1 討論指南

## 本週 1-on-1 聚焦

個人小專題的第一次深入 debug + 方向確認。

## 流程（60 分鐘）

| 段落 | 時間 | 內容 |
|------|------|------|
| 回顧 | 5 分鐘 | 上週作業 |
| 專題 debug | 35 分鐘 | 看學生的第一版程式，一起找問題 |
| 下週計畫 | 15 分鐘 | 確認下週（視覺化週）學生要畫哪些圖 |
| 備用 | 5 分鐘 | |

## 專題 debug 引導問題

**理解題目**：
- 「你的程式在模擬什麼？說給我聽。」（讓學生說，不要老師先說）
- 「哪個部分是隨機的？」

**確認邏輯**：
- 「`simulate_once()` 做了什麼？回傳了什麼？」
- 「你迴圈跑了幾次？每次做什麼？」

**改進方向**：
- 「結果看起來合理嗎？如果我把這個參數改大，你預測結果會怎樣？」

## 下週視覺化計畫

確認學生知道下週要畫什麼圖：
- 結果分佈 → 直方圖
- 隨時間變化 → 折線圖
- 兩個變數的關係 → 散點圖

「你的題目最適合用哪種圖？為什麼？」
